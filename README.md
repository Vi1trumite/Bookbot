# Bookbot
A Python CLI program that analyzes the text of a book and generates a report

# Features
- Counts the total number of words in the document
- Counts the frequency of each letter (case sensitive)
- Accepts a file path a command line argument via sys.argv
- Exits cleanly with an error message if no file is provided

# Project Structure
- main.py  entry point, handles argument parsing and output
- stats.py contains the word and letter count functions# bookbot
